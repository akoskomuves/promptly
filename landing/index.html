Know what your AI
coding sessions cost

Capture, analyze, and review AI conversations tied to development work. Track tokens, duration, and patterns.

$ npm i -g @getpromptly/cli

Features

Local-first

Everything on your machine. SQLite database, no account needed, fully offline.

Multi-tool support

Works with Claude Code, Gemini CLI, Codex CLI, Cursor, Windsurf, and VS Code/Copilot.

Ticket tracking

Tag sessions to JIRA/Linear tickets. See how much each feature costs in tokens.

Built-in dashboard

View sessions, token usage, and conversation history at localhost:3000.

How it works

promptly init # auto-detect & configure tools
promptly start TICKET-1 # start logging
# ... work with any supported AI coding tool ...
promptly finish # save session data
promptly serve # open dashboard
